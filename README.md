# 🧠 Memory Game - Brawl Stars

Juego de memoria temático de **Brawl Stars** con 24 cartas (12 parejas). Pon a prueba tu memoria antes de que se acabe el tiempo.

## 🎮 Cómo jugar

- Haz clic en las cartas para voltearlas
- Encuentra las 12 parejas de personajes de Brawl Stars
- Completa el juego antes de que el temporizador llegue a cero

## ⏱️ Niveles de dificultad

| Nivel      | Tiempo |
|------------|--------|
| Easy       | 3 min  |
| Medium     | 2 min  |
| Hard       | 1 min  |
| Extreme    | 40 seg |
| Impossible | 20 seg |

## 🕹️ Funciones

- **Start Game** — Inicia la partida y activa el temporizador
- **Pause Game** — Pausa la partida
- **Show All / Hide All** — Revela u oculta todas las cartas
- **Match All** — Empareja automáticamente todas las cartas
- **Selector de nivel** — Elige la dificultad antes de empezar

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
